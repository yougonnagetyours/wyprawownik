# Przygotowania do wyprawy — Balkany Maj 2026

Wyprawa: 16–24 maja 2026 | ~3000 km | Serbia, Czarnogóra, Albania
Auto: Nissan Terrano 2, 2.7 TDI, 4x4, rocznik 2000, ~200 tys. km

- [ ] Assistance drogowe — rozszerzone na Bałkany (holowanie, pomoc w trasie)

## 2. Sprzęt awaryjny i narzędzia

### Obowiązkowe (prawo + zdrowy rozsądek)
- [ ] Trójkąt ostrzegawczy
- [ ] Kamizelki odblaskowe — 2 szt. (w niektórych krajach wymagane dla każdego pasażera)
- [ ] Apteczka samochodowa
- [ ] Gaśnica
- [ ] Koło zapasowe (pełnowymiarowe!) + klucz do kół + lewarek
- [ ] Żarówki zapasowe (w Serbii obowiązkowe)

### Narzędzia i recovery
- [ ] Skrzynka z narzędziami: klucze płaskie/oczkowe, śrubokręty, kombinerki, klucz nastawny
- [ ] Linka holownicza / pas ratunkowy
- [ ] Szekle (2 szt.)
- [ ] Kompresor 12V do opon
- [ ] Taśma duct tape — naprawia wszystko
- [ ] Opaski zaciskowe (różne rozmiary)
- [ ] Drut wiązałkowy
- [ ] Saperka / łopatka składana
- [ ] Bezpieczniki zapasowe (komplet do Terrano)
- [ ] Olej silnikowy zapasowy (1L)
- [ ] Płyn chłodniczy zapasowy (1L)

### Opcjonalne (nice to have)
- [ ] Hi-Lift jack (do wyciągania z błota/piasku)
- [ ] Pasy do piaskowania / traction boards (Albania — drogi gruntowe)
- [ ] Szpilki do naprawy opon (tire plug kit)

## 3. Sprzęt biwakowy i campingowy

### Nocleg
- [ ] Zabudowa bagażnika — w trakcie (deska + materac = łóżko w aucie)
- [ ] Materac składany — budżet ~300 zł
- [ ] Śpiwory x2 (komfort min. 5°C — maj w górach Czarnogóry bywa chłodny nocą)
- [ ] Poduszki podróżne
- [ ] Koc / pled (dodatkowe ciepło + siedzenie na zewnątrz)
- [ ] Namiot — backup na wypadek gdyby w aucie nie wyszło, albo na kemping
- [ ] Moskitiera / siatki na okna (Albania, wieczory nad wodą)

### Kuchnia
- [ ] Kuchenka turystyczna (gazowa, małe naboje)
- [ ] Naboje gazowe — 6-8 szt. na tydzień
- [ ] Garnek + patelnia (kompaktowe, turystyczne)
- [ ] Kubki, talerze, sztućce — po 2 komplety
- [ ] Nóż turystyczny / wielofunkcyjny
- [ ] Otwieracz do puszek/butelek
- [ ] Kanister na wodę (10-20L) — pitna + do mycia
- [ ] Deska do krojenia (mała)
- [ ] Worki na śmieci
- [ ] Gąbka + płyn do mycia naczyń
- [ ] Zapalniczka / zapałki

### Komfort
- [ ] Krzesła składane x2 (turystyczne, lekkie)
- [ ] Stolik składany (mały)
- [ ] Lampa / latarnia LED (na biwaku wieczorem)
- [ ] Latarka czołowa — do chodzenia nocą, napraw
- [ ] Plandeka (tarp) — płachta ~3x3m, rozciągasz na linkach między drzewami = daszek od deszczu/słońca (~30-50 zł)
- [ ] Sznurek / lina (do tarpa, suszenia prania itp.)

## 4. Rzeczy osobiste i pozostałe

### Dokumenty i pieniądze
- [ ] Dowody osobiste / paszporty
- [ ] Karta EKUZ (Europejska Karta Ubezpieczenia Zdrowotnego)
- [ ] Ubezpieczenie turystyczne (KL + NNW) — Albania to kraj spoza UE!
- [ ] Gotówka: EUR (Serbia, Czarnogóra, Albania przyjmują) + LEK albański (małe kwoty)
- [ ] Karta płatnicza (Revolut/Wise — bez prowizji za przewalutowanie)

### Higiena i zdrowie
- [ ] Ręczniki szybkoschnące x2
- [ ] Kosmetyczka (żel, szampon, pasta, szczoteczka)
- [ ] Krem z filtrem UV — maj = mocne słońce
- [ ] Spray na komary/kleszcze — Albania, biwakowanie
- [ ] Apteczka osobista: ibuprofen, węgiel, elektrolity, plastry, bandaż, środek odkażający
- [ ] Papier toaletowy — to się szybko przydaje na dziko

### Nawigacja
- [ ] Mapy offline w telefonie (maps.me / Google Maps offline)
- [ ] Powerbank (duży, 20000+ mAh)
- [ ] Ładowarka samochodowa USB (12V → USB)
- [ ] Przetwornica 12V → 230V (ładowanie laptopa, kamery)

## 5. Sprzęt do nagrywania vloga

Celem jest pamiątka + vlog — nie profesjonalna produkcja. Setup ma być lekki, prosty, budżetowy.

### Sprzęt Michała (mamy)
- [x] Telefon: **Xiaomi 15T Pro** — kamera top klasa, stabilizacja, 4K
- [x] Statyw: **Gorillapod** — elastyczny, można przymocować do barierki, gałęzi, auta
- [x] Mikrofon: **Rode VideoMic GO II HELIX + deadcat** — świetny mikrofon kierunkowy, deadcat na wiatr = kluczowy na Bałkanach

### Do dokupienia
- [ ] Uchwyt na telefon do auta (na szybę/kratkę) — ujęcia z jazdy, ~30-50 zł
- [ ] Karty pamięci zapasowe (microSD 64-128GB) — ~30-60 zł
- [ ] Powerbank (duży — 20000 mAh+) — jeśli nie masz
- [ ] Woreczki strunowe / etui wodoodporne (kurz, deszcz, Albania) — ~10-20 zł

### Opcjonalne ulepszenia (jeśli budżet pozwoli)
- [ ] GoPro Hero (używana 9/10/11, OLX ~600-1000 zł) — fajne ujęcia z jazdy, off-road, wodoodporna
- [ ] Przyssawka GoPro do szyby — ~50 zł (mega klimat ujęć z drogi)
- [ ] Gimbal do telefonu (DJI OM / Zhiyun Smooth ~400 zł) — spacerowe ujęcia bez drgań

### Tipy filmowe (z doświadczeń podróżników)
- **Busem Przez Świat** zaczynali od samego GoPro — potem doszła Panasonic GH4, Sony A7S, dron DJI, gimbal Ronin S. Ale ich najpopularniejsze filmy były kręcone GoPro!
- Nagrywaj codziennie choćby 5 min — łatwiej zmontować niż odtwarzać z pamięci
- Ujęcia z jazdy (dashcam/GoPro na szybie) dają super klimat
- Poranne i wieczorne światło = najlepsze kadry
- Dźwięk jest ważniejszy niż obraz — nawet tani mikrofon robi ogromną różnicę vs. wbudowany
